# LabHero

Deploy rápido do app LabHero usando GitHub Pages.

## Como funciona
- O GitHub Actions publica todo o conteúdo do repositório na branch `work`.
- Quando você fizer push, o site será atualizado automaticamente.

## Próximos passos
1. Substitua o conteúdo do `index.html` pelo build do app.
2. Acesse **Settings → Pages** e selecione "GitHub Actions" como fonte.
3. Faça o push e aguarde o deploy concluir.
